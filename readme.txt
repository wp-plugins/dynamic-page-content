Dynamic Page Content creates a shortcode that you can use to get the value of URL parameters and use them to create dynamic page content

== Description ==

This Plugin "Dynamic Page Content" was created to build WordPress Pages with dynamic content that is generated by URL parameters. It's main purpose is to create Google AdWords Ads with landingpages containing parameters in this form: yourwebsite.com/some-directory/?prmtr=VALUE. You can use this shortcode [DynamicPageContent] to output VALUE from the anywhere you want on a WordPress page. With this method you can create a very high number of Google AdWords Ads with automatically created and very compelling content for a very high conversion rate.

Major features in Dynamic Page Content include:

* Creates a Shortcode that you can use on any page in your Wordpress installation
* Replaces the "Shortcode" with the value from the URL Parameter called "prmtr"

== Installation ==

Upload the plugin to your blog, Activate it.
